# ExilesReagentHelper

A companion plugin for [ExileCore2](https://github.com/exApiTools) that
builds a clean game-state snapshot — vitals, buffs, skills, flasks, and nearby monsters — for
use with ReAgent-style automation. 

## Project layout

| File / folder | Purpose |
|---------------|---------|
| `State/` | Game-state snapshot — vitals, buffs, skills, flasks, nearby monsters. |
| `ExilesReagentHelper.cs` / `ExilesReagentHelperSettings.cs` | Plugin entry point and settings. |
